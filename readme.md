### 설명

- 영화 정보 및 알고리즘 추천 사이트
- 기획의도 : 21세기에 20세기를 체험하는 느낌을 주려고 했음. 즉, 풍부한 정보 속에서 영화를 고르는 것이 아닌, 자신의 기호대로 영화를 선택할 수 있는 기회를 주고자 했다. 

### 목표

- 모던한 느낌의 페이지 구성
- 다른 페이지에서는 느낄 수 없는 특별한 사용자 경험
- 영화 데이터 제공 및 유저 데이터 기반 알고리즘 추천 서비스
